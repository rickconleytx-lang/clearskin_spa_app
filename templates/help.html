Help & Quick Reference

Daily Workflow

Book Appointment

  • Go to Add Appointment
  • Search client by last name
  • Select client
  • Select service
  • Choose date and time
  • Click Save Appointment

Daily Schedule

  • Open the calendar
  • Select the date
  • Review appointments
  • Use Pre-Session Intake and Forms
  • Edit appointment if needed

Finish Session

  • Click Finish Session
  • Enter payment information
  • Complete post-session wrap-up
  • Optionally book next appointment

Appointments Page

  • Today = today’s appointments only
  • Filter = date range search
  • Show All Appointments = full history

Reschedule an Appointment

  • To reschedule an appointment please click Edit from the appointment on the Calendar
  • or from the Appointments page, click on the client and appointment you wish to rechedule
  • and click on Edit

Client Search

  • Search by last name
  • Select client from dropdown
  • If not found, add as new client

Troubleshooting

  • If client is missing, verify spelling of last name
  • If appointment is missing, check date filters
  • If service dropdown is incomplete, verify service_name_types
