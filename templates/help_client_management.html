Client Management Help

Using the Appointments Page

The Client Management page is your Client everything page. Anything and Everything you wish to accomplish with a client can be completed on this page.

This page is divided into two sections. The top part (Client Management Navigation Panel) and the Client Action List.

Client Management Navigation Panel

Add New Client Button

If you know for sure that you are going to Add a New Client to your database, then click this button. You will be redirected to the Add New Client Form. Here you will enter client's name, Phone, email, birthday, address, emergence contact , referral and three notes fields.

Go To Calendar

You're one click away from your calendars page, click this button to return to your Calendar Page.

Clients Home

This button will redirect you to the Clients Home Page. Clients Home Page has Birthday Offers information, Add Client option and the Client History and Full Record View/Edit. Client's full health record, vist records and all other details can be found on this page.

Need to finish

  • Clicking Filter will display the selected date range in the Appointment List.

Today

  • Clicking Today will display today's appointments in the Appointment List.

Show All Appointments

  • Displays all appointments ever scheduled in the Appointment List.

Add Appointment

  • Opens the Daily Schedule page so you can enter details for a new appointment.

Appointment List View

The Appointment List View displays all appointments returned by the current search. Each line contains appointment details as well as buttons for Edit, Add Income, and Cancel.

Appointment Line

Left to right:

  • ID - each appointment has a unique appointment ID
  • Date of appointment
  • Time of appointment
  • Client name - Last Name, First Name
  • Service scheduled
  • Appointment status - Completed / Booked
  • Actions
    • Edit Button - Click to edit or reschedule the appointment
    • Add Income Button - Click to finish the session, enter payment information, and complete the post-a$
    • Cancel Button - Click to cancel the appointment
