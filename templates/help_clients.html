Clients Help

Using the Clients Page

The Appointments page has two sections: the Search section and the Appointments List. The Search section is used to change the date range for the appointments shown in the Appointment List.

When the Appointments page is opened, it will show the current day's appointments in the Appointment List. If the day or date range you are looking for is already displayed, there is no need to use the Search section. If the day or date range you want is not displayed, you will need to use Search.

Using Search

From - To (Date Range)

Use the Date Range Search to search for either a single day or a range of dates.

To search for only one day, click inside the From box and use the Date Picker to select the date. Then click in the To box and use the Date Picker to select the same date. Finally, click the Filter button.

To search for a range of dates, click inside the From box and use the Date Picker to select the beginning date. Then click in the To box and use the Date Picker to select the ending date of your date range. Finally, click the Filter button to display the scheduled appointments within that date range.

Filter

  • Clicking Filter will display the selected date range in the Appointment List.

Today

  • Clicking Today will display today's appointments in the Appointment List.

Show All Appointments

  • Displays all appointments ever scheduled in the Appointment List.

Add Appointment

  • Opens the Daily Schedule page so you can enter details for a new appointment.

Appointment List View

The Appointment List View displays all appointments returned by the current search. Each line contains appointment details as well as buttons for Edit, Add Income, and Cancel.

Appointment Line

Left to right:

  • ID - each appointment has a unique appointment ID
  • Date of appointment
  • Time of appointment
  • Client name - Last Name, First Name
  • Service scheduled
  • Appointment status - Completed / Booked
  • Actions
    • Edit Button - Click to edit or reschedule the appointment
    • Add Income Button - Click to finish the session, enter payment information, and complete the post-a$
    • Cancel Button - Click to cancel the appointment
